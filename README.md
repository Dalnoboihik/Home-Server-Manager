🏠 **Home Server Manager v2.5.0**
📖 Общее описание
Home Server Manager — это мощный веб-интерфейс для управления домашним или офисным сервером на базе Windows. Программа позволяет удаленно контролировать состояние системы, управлять файлами и отслеживать нагрузку на сервер через любой современный браузер.

Программа разработана на Python с использованием фреймворка Flask и предоставляет интуитивно понятный интерфейс в темной цветовой гамме, адаптированный для работы на различных устройствах (ПК, планшеты, смартфоны).

🔧 Установка и запуск
Быстрый старт:
Установите Python 3.8+

Скачайте проект в папку **C:\Users\Gorbunovsk\Desktop\home-server-manager**

Установите зависимости:

cmd
pip install flask flask-login psutil werkzeug
Запустите сервер:

cmd
python working_server.py
Откройте браузер: http://127.0.0.1:5000

Войдите: логин admin, пароль admin123
